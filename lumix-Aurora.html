Lumix Aurora

Reliable and versatile AI image generation model, built on proven Stability AI foundations.

Model Overview

Lumix Aurora AI is a trusted model based on the widely adopted Stability AI framework. It offers a balanced mix of quality, speed, and resource efficiency, making it suitable for general-purpose image generation needs.

It supports a wide range of styles, from photorealistic to artistic renditions, and is optimized for cloud deployment at scale.

API Endpoint

POST https://api.lumix.ai/v1/generate

Use lumix-stability-ai as the model parameter in your request.

Request Headers

  • Authorization: Bearer <your_api_key>
  • Content-Type: application/json

Request Body Parameters

{
  "prompt": "string (required)",
  "model": "lumix-stability-ai",
  "size": "string (optional, default: 512x512)",
  "steps": "integer (optional, default: 25)",
  "guidance_scale": "float (optional, 1.0-15.0, default: 7.0)",
  "seed": "integer (optional)",
  "batch_size": "integer (optional, default: 1, max: 8)"
}

Sample API Request

{
  "prompt": "A serene landscape with mountains and a lake at sunrise",
  "model": "lumix-stability-ai",
  "size": "512x512",
  "steps": 25,
  "guidance_scale": 7.0,
  "seed": 33445,
  "batch_size": 2
}

Sample API Response

{
  "status": "success",
  "request_id": "req_stab789",
  "images": [
    "https://cdn.lumix.ai/generated/landscape_sunrise_789_1.jpg",
    "https://cdn.lumix.ai/generated/landscape_sunrise_789_2.jpg"

  "model_used": "lumix-stability-ai",
  "processing_time": "2.7s"
}

Authentication & Rate Limits

  • API key required via Authorization header.
  • Rate limit: 50 requests per minute per API key.
  • Batch requests count per image.

Error Handling

Standard HTTP error codes apply:

  • 401 Unauthorized — Invalid or missing API key.
  • 429 Too Many Requests — Rate limit exceeded.
  • 400 Bad Request — Payload issues.

Refer to the JSON message for detailed info.

Tips & Best Practices

  • Use moderate steps and guidance_scale for balanced quality and speed.
  • Leverage batch requests to optimize throughput.
  • Use seed for reproducibility.
  • Ideal for diverse image generation needs across styles.
