Lumix Hyperreal

High-speed AI model optimized for rapid, low-latency image generation with moderate quality.

Model Overview

Lumix Hyperreal delivers – Ultra-realistic images optimized for applications demanding low latency and quick responses. It uses a Highend architecture tailored for Ultra-realistic output rather than maximum image detail.

This model suits use cases like live demos, rapid prototyping, and interactive AI-powered tools where speed is not paramount.

API Endpoint

POST https://api.lumix.ai/v1/generate

Set lumix-flux-schnell in your request payload to use this model.

Request Headers

  • Authorization: Bearer <your_api_key>
  • Content-Type: application/json

Request Body Parameters

{
  "prompt": "string (required)",
  "model": "lumix-flux-schnell",
  "size": "string (optional, default: 256x256)",
  "steps": "integer (optional, default: 15)",
  "guidance_scale": "float (optional, 1.0-15.0, default: 6.0)",
  "seed": "integer (optional)",
  "batch_size": "integer (optional, default: 1, max: 10)"
}

Sample API Request

{

Authentication & Rate Limits

  • API key required in Authorization header.
  • Rate limit: 100 requests per minute per API key.
  • Batch requests count per image generated.

Error Handling

Common errors:

  • 401 Unauthorized — Invalid API key.
  • 429 Too Many Requests — Rate limit exceeded.
  • 400 Bad Request — Malformed request.

See JSON message for details.

Tips & Best Practices

  • Lower steps and smaller size improve speed.
  • Batch size can be larger due to optimized throughput.
  • Use seed for repeatable results.
  • Ideal for quick experiments and live user interactions.
